# Задание 1

## Задача 1

### Текст задачи

Найти сумму делителей данного натурального числа. 

### Алгоритм решения

1)Ввод натурального числа N

2)Инициализация: начальное значение делителя Current = 1, накопленная сумма Acc = 0

3)Проверка Current от 1 до N:

  -Если Current > N → завершить, сумма = Acc
  
  -Если N делится на Current (N mod Current = 0) → добавить Current к сумме
  
  -Если не делится → сумму не менять
  
  -Перейти к следующему Current (Current + 1)
  
4)Вывод полученной суммы

# Задание 2

## Задача 1

### Текст задачи

Если сумма элементов списка отрицательна, вывести –1, если положительна —
1, если равна нулю, то 0. 

### Алгоритм решения

1)Ввод списка целых чисел
2)Вычисление суммы элементов списка:
    -Если список пустой → сумма = 0
    -Если список не пустой → сумма = первый элемент + сумма оставшихся
3)Определение знака суммы:
    -Если сумма < 0 → знак = -1
    -Если сумма > 0 → знак = 1
    -Если сумма = 0 → знак = 0
4)Вывод знака
